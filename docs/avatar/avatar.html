Avatar

An avatar is used to represent the user's profile picture or their initials.
Image Avatar
You can create image avatar using the avatar class in img tag
Size
You can change the size of avatar by adding the following classes avatar-sm, avatar-md, avatar-lg, avatar-xl to the image tag
